Posted on 2018-05-15 | In 使用技巧

快捷键

搜索

作用 mac win
搜索类 cmd + n ctrl + n
全局路径搜索 shift + ctrl + f shift + ctrl + f
按照symbol查找 shift + cmd + alt + n
按照文件名查找 shift + cmd + n
跳转下个高亮 f3

使用inject language输入特殊字符串

mac：先使用alt+enter弹出inject language视图，在Inject language or reference中按回车选择需要输入ed字符串，再次按下alt+enter即可看到

快捷输入System.out.print

输入sout后回车

自动收尾，补全括号：

mac：cmd + shift + enter

弹出action框

mac：shift+cmd+A

快速选择多个相同元素

mac：选择一个元素alt+G，再次按下alt+G选择下一个相同元素，按下alt+cmd+G选择全部相同元素

gennerate选框：

mac：ctrl + enter

win：ctrl + insert

代码重构：

用途 mac win
重命名 shift + f6
取出变量 cmd + alt + v
将构造提取成函数 cmd + alt + m
代码格式化 cmd + alt + l

代码助手：

用途 mac win
查看参数信息 cmd + p
弹出函数定义 shift + cmd + i
跳转错误处 f2
显示错误信息 cmd + f1
查看文件结构 cmd + f12
查看函数注释 ctrl + j

项目设置Project Struct

project

  • Project name: 定义项目名称
  • Project sdk: 设置项目使用的JDK
  • Project language level: 限定项目编译检查时最低的JDK特性
  • Project complier output: 项目默认的编译输出总目录

Modules

每个子项目都对应了Sources、Paths、Dependecies三大配置选项

  • Sources：显示项目的目录资源、那些是项目部署的需要的目录
  • Paths：指定项目的编译输出路径
  • Dependecies：项目的依赖

Libraries

显示所添加的jar包

Facets

Artifacts

编译后的Java类，Web资源整合，用以测试、部署等工作。
